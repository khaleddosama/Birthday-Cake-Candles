# Birthday-Cake-Candles

---

##  Overview
This project implements the Birthday Cake Candles problem using recursive and non-recursive approaches in C.

The project focuses on algorithm implementation and time complexity analysis.

---

##  Objectives
- Implement recursive and iterative solutions
- Compare recursion and iteration performance
- Analyze time complexity

---

##  Key Features
- Recursive implementation
- Non-recursive implementation
- Maximum value counting
- Complexity analysis O(n)
